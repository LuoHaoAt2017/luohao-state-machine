以H2O为例

定义状态
1. 固态【冰】
2. 液态【水】
3. 气态【气】

定义变化
融化【固态 -> 液态】
蒸发【液态 -> 气态】
液化【气态 -> 液态】
凝固【液态 -> 固态】
升华【固态 -> 气态】
凝华【气态 -> 固态】

监听变化
融化开始 融化结束
蒸发开始 蒸发结束
液化开始 液化结束
凝固开始 凝固结束
升华开始 升华结束
凝华开始 凝华结束
